# exipe2

Exipe is an API for information extraction from presentation document.

The API is currently in developement, bugs are likely to occur.

Implemented features : 
*Title extraction
*Slide body text extraction
*Named entities extraction (buggy)
*URLs extraction

Not yet implemented :
*Structure extraction

For now the API works only with Office Open XML Presentation files (PPTX). It uses python-pptx and NLTK.
Documentation is not written
